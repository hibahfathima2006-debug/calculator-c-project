# Calculator using C

## Description
A simple calculator program in C using switch-case statements.
Supports addition, subtraction, multiplication, and division.
Handles division by zero error.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Error handling for division by zero

## How to Run
1. Compile: `gcc calculator.c`
2. Run: `./a.out`

## Technologies Used
- C Programming
